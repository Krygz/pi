Instruções de Uso
  1. Clique em "Abrir Console H2"
  2. Preencha os dados de conexão acima
  3. Clique em "Connect"
  4. Explore as tabelas: USUARIO e ORCAMENTO
Consultas Úteis
Ver todos os usuários:
SELECT * FROM USUARIO;

Ver todos os orçamentos:
SELECT * FROM ORCAMENTO;

Orçamentos com usuários:
SELECT o.*, u.nome_usuario FROM ORCAMENTO o JOIN USUARIO u ON o.usuario_id = u.id;
